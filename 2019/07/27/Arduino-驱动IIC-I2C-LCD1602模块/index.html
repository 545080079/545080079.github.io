Arduino----I2C LCD1602模块

I2C

I2C又称IIC，即Inter-Integrated Circuit(集成电路总线）。
多个芯片可以连接到同一总线结构下，同时每个芯片都可以作为实施数据传输的控制源。
I2C串行总线一般有两根信号线，一根是双向的数据线SDA，
另一根是时钟线SCL。所有接到I2C总线设备上的串行数据SDA都接到总线的SDA上
，各设备的时钟线SCL接到总线的SCL上。

##连接
I2C LCD1602集成了4个引脚，我采用的是PCF85741驱动芯片。

LCD1602<—>Arduino UNO开发板 连接方式如下：

GND <—> GND

VCC <—> 5V

SDA <—> SDA

SCL <—> SCL

##Arduino代码——lcd1602IIC.ino

#include <Wire.h>                   //I2C库文件
#include <LiquidCrystal_I2C.h>      //I2C 1602库文件，可以通过自带的库管理器添加

LiquidCrystal_I2C lcd(0x27,16,2);   // LCD地址是0x27，设置为16字/行，共2行

 void setup() {
  lcd.init();
  lcd.backlight();
  lcd.print("Author:Luoyutao");
  lcd.setCursor(0,1);               //新的一行
  lcd.print("www.lyt0617.cn");

}

void loop() {


}
